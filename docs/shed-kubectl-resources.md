# shed-kubectl-resources

Print list of resources available for `shed-kubectl-apply` and `shed-kubectl-delete`.

## SYNOPSIS

```bash
shed-kubectl-resources
shed kubectl-resources
```

## EXAMPLES
```bash
$ shed-kubectl-resources
hello-world
```
