# NAME
shed-version - print the current version of Shed.

# SYNOPSIS
**shed-version**\
**shed version**

# DESCRIPTION
Print the current version of Shed.

# EXAMPLES
**shed-version**\
\ \ \ \  shed version: 1.23.0
